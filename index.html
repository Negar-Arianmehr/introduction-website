GiT Wedding Project

Hi dear GiT spouses from now and past years, Before the Summer, we came up with the idea, initiated by Megha Vaidya, to organize a GiT exhibition of the various wedding celebrations around the world .. traditions, functions, rituals, etcetera .. current and from the past.. This has been triggered by our regular GiT exchanges of life experiences & life events in various GiT sessions over the past years. It then appears that there was so much to share and show about the weddings of GiT spouses from all over the world .. so many differences, so many common grounds as well, so many colors, customs and meaningful memories. That was the starting point for us to think of ways how to share all these beautiful and inspirational stories with the outer world in Eindhoven as well. Well .. now this idea has been elaborated in an extra GiT committee .. the 'GiT Wedding Prep group', in which are participating so far: Megha, Negar, Nandita, Gabriela, Komala & me of course. To start with, this GiT committee has created a survey to collect stories & ideas for our GiT International Wedding exhibition, planned in May/June 2021 @ the Pennings Photography Gallery (more detailed info will follow later). If you're also interested in this project, then pls fill in the survey by clicking on the link below, to share your info, experiences, ideas and also if you like to get more specifically involved in prepping this exhibition. The outcome will give us more ideas how to organize and 'dress' the planned exhibition. So this first step will help us proceed forward to this exciting new extra GiT activity .. very welcome!! Thx a lot in advance .. we'll keep you posted, Carola, Megha, Negar, Nandita, Gabriela & Komala THX IN ADVANCE .. WE'LL KEEP YOU POSTED .. üíçüíç
